# PPI Network Analysis for Diabetic Wound Healing

## Overview
This project focuses on analyzing Protein-Protein Interaction (PPI) networks related to diabetic wound healing. The goal is to identify key proteins, interaction patterns, and biological insights that can help in understanding the healing process in diabetic conditions.

---

## Objectives
- Construct and analyze PPI networks
- Identify key proteins (hub proteins) involved in wound healing
- Perform network topology analysis
- Visualize biological interaction networks
- Extract meaningful biological insights using computational methods

---

## Dataset
- Protein interaction data obtained from public biological databases (e.g., STRING)
- Filtered for proteins relevant to diabetic wound healing pathways

---

## Methodology
1. Data collection from biological databases
2. Preprocessing and cleaning interaction data
3. Construction of PPI network using graph-based approach
4. Network analysis using centrality measures:
   - Degree Centrality
   - Betweenness Centrality
   - Closeness Centrality
5. Visualization of network graphs
6. Interpretation of biological significance

---

## Technologies Used
- Python
- NetworkX
- Pandas
- Matplotlib
- Bioinformatics databases (STRING / others)

## Key Results
- Identification of important hub proteins in diabetic wound healing pathways
- Visualization of interaction clusters
- Network-based insights into biological relationships
